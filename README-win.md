Add to PATH:
  Open expolorer
  Right-click on This PC
  Choose properties
  Click on Advanced system settings
  On tab Advanced, click on "Environment Variables..." button at bottom
  Add %PROGRAMFILES%\Oracle\VirtualBox to PATH...
